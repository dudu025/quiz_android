Este projeto é um quiz mobile onde o usuário pode criar uma conta, fazer login e editar suas informações, tudo salvo localmente usando Room. O aplicativo também registra e armazena os rankings no banco de dados interno do dispositivo.
As perguntas e respostas do quiz são carregadas pela internet por meio de uma API externa.
